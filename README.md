# Span-based Syntactic Feature Fusion for Aspect Sentiment Triplet Extraction

## Setup

This code runs Python 3.8 with the following libraries:

+ Pytorch 2.0.1
+ Transformers 4.31.0
+ spacy 3.7.2

## Get start

1. Build vocabulary

   ```
   python prepare_vocab.py
   ```

2. Training

   Go to Corresponding directory and run main.py:

   ``` 
   python main.py
   ```
